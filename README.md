# 无尽仙台 (Endless Sendai)

欢迎来到 **《无尽仙台》** 项目仓库！

《无尽仙台》是一款的修仙题材回合制构筑卡牌游戏。玩家将在游戏中的“登仙台”内反复轮回，通过积累记忆与道行（剑意），寻找破局飞升之法。

## 🎮 当前状态

- **阶段**：前期开发中
- **进度**：
  - 各类基础 UI 结构搭建中（主菜单、选项、选角、战斗场、地图等）。
  - 核心设计文档（包含角色、怪物、卡牌机制、地图及遗物系统）大量产出，内容详尽。
  - 主要战斗交互及表现层系统还在逐步迭代中。

## 🛠 技术栈

- **游戏引擎**: Godot 4.3+ (GDScript)
- **平台**: Windows (PC)
- **UI 风格**: 基于 Godot 控件的 GUI (拉伸模式 `canvas_items + expand`)
- **多语言**: 通过 `TranslationServer` 框架集成管理（中文 `zh_CN`、英文 `en`）

## 📁 目录结构说明

```text
├── assets/       # 音效、美术贴图等各类静态资源

```

## ⚔️ 游戏核心特色概览

详细玩法与设定，请阅读 `需求文档/` 内的文件。

1. **三大资源核心战斗**
   - **生命(HP) / 灵气(Energy)**：基础战斗流通货币；
   - **道行/剑意(Sword Intent)**：跨回合保留的核心独特资源，允许玩家在游戏进程中积蓄极大的破坏力或应对机制。
2. **多形态地图路线探索**
   - 分为普通战斗、精英遭遇、休憩(篝火)、黑市(商店)及遭遇(事件)。玩家从底向上逐步推进，三重天阶跃最后抵达结局。
3. **卡牌构建流派**
   - 丰富的词缀效果与“术法、秘法、道法”三系卡牌分支，包含四种稀有度阶层。

## 🚀 运行与环境配置指南

1. 把本库克隆或下载到本地。
2. 下载 [Godot Engine](https://godotengine.org/download) （推荐使用最新稳定版 4.3 以上；项目内可能已提供独立运行的exe格式）。
3. 启动 Godot 引擎，在项目列表右上角点击 `导入(Import)`，选择本仓库的 `project.godot` 文件导入。
4. 双击打开项目后，点击右上角的“▶ 运行”按钮（快捷键 `F5`）即可启动游戏的主菜单界面。

## 📋 协作者开发规范

本项目注重**自动化测试**及**文档强同步**，请所有开发者遵守以下工作流闭环：

1. **查阅需求文档**：每开发一个新的场景、机制前，必须到 `需求文档/` 中查阅相关设计。
2. **编写功能代码**：确保其结构与现存逻辑能平滑整合。如果设计改变，**第一时间同步更新对应的需求文档**，确保开发与文档时刻对齐。
3. **完成验收测试**：任何逻辑及模块实装必须附带测试代码！
   - 测试脚本存放在 `tests/suites/` 中，继承于项目自带的 `TestSuite`。
   - 完成后前往 `tests/TestMain.gd` 的执行区进行套件注册。
   - 打开终端/目录双击执行项目根目录的 `run_tests.bat`，查阅输出结果，**所有的测试全部 Passed (通过) 才算结束当前任务**。
4. 注释规范化：编写复杂逻辑时确保逐行写清注释流，让任何人阅读代码能够快速理清每一步的作用。
