# 无尽仙台

《无尽仙台》是一款修仙题材的回合制构筑卡牌 Roguelike。玩家困于"登仙台"的无尽轮回之中，每次死亡后带着记忆重头再来，以无数次的失败为代价，寻找斩破天道牢笼的唯一破局之法。

---

## 核心玩法

### 双资源战斗系统

战斗由两种独立资源驱动，互相制约，形成持续的成长博弈：

| 资源 | 说明 |
|------|------|
| **灵力** | 每回合开始回复（默认 +3），回合结束消散当前值的 1/10。前期灵力是瓶颈，决定每回合能打几张牌。 |
| **道慧** | 每回合开始重置至上限（默认 10），回合结束**清零**，不可跨回合保留。后期灵力提升后，道慧成为真正的强度上限。 |

> 灵力是"行动次数"，道慧是"每回合的强度配额"。两个资源在不同阶段轮流卡住玩家的上限。

### 防御层级

伤害按优先级依次击破：**护体（永久罡气）→ 生命值**

### 卡牌体系（剑修）

- **三系卡牌**：术法（攻击）/ 秘法（防御运转）/ 道法（全局被动，打出后移出本局）
- **四档稀有度**：黄品 55% / 玄品 30% / 地品 10% / 天品 5%
- **词缀机制**：耗尽 / 保留 / 溃散 / 本源 / 禁锢

### 地图结构

三重天（每重 15 节点 + Boss）推进，节点类型包括：普通战斗、精英战斗、篝火、黑市、执念事件、天门 Boss。单向不可逆，Boss 依次为剥皮仙君 → 无面司命 → 收割者 → 隐藏终局天宫之主。

## 技术栈

- **引擎**：Godot 4.3+（GDScript）
- **平台**：Windows
- **多语言**：中文 / 英文，通过 `TranslationServer` 管理

---

## 目录结构

```
scenes/      # .tscn 场景文件
scripts/     # .gd 脚本
tests/       # 自动化测试
i18n/        # 多语言翻译文件
theme/       # 主题资源
需求文档/    # 完整设计文档
```

---

## 运行方式

1. 安装 [Godot 4.3+](https://godotengine.org/download)
2. 在 Godot 中导入 `project.godot`
3. 按 `F5` 运行

---

## 开发规范

每完成一个功能模块，必须同步更新对应需求文档，并在 `tests/suites/` 下编写测试，执行 `run_tests.bat` 全部通过后方可继续。
