# 黑市商品 V2

黑市使用局内货币“灵石”。V2 保留卡牌、宝物、删牌、升级服务的既有规则，只重构背包消耗品。

宝物清单以 `需求文档/宝物.txt` 为准；本文件只维护黑市刷新规则、背包物品与服务接口。

## 一、商品刷新

| 区域 | 每次进货 | 规则 |
|---|---:|---|
| 卡牌 | 3 张 | 从 `CardDatabase` 抽取，按当前地图层数分早/中/后期稀有度池，不重复 |
| 物品 | 4 个 | 统一包含丹药、符箓、阵盘，按 `min_floor` 过滤，不重复；临时加货可变为 5 个 |
| 宝物 | 2 个 | 不展示本局已拥有宝物，同一商店内不重复；具体数据见 `需求文档/宝物.txt` |
| 服务 | 删除/升级 | 删除卡牌每次进入黑市最多 1 次；升级可多次，只能升级未升级卡 |

层数分段：

| 阶段 | 层数 | 卡牌稀有度池 | 商品解锁 |
|---|---|---|---|
| 早期 | 1-5 | 黄品为主，少量玄品 | `min_floor <= 5` |
| 中期 | 6-10 | 黄/玄为主，少量地品 | `min_floor <= 10` |
| 后期 | 11+ | 玄/地为主，少量天品 | `min_floor <= 当前层` |

## 二、价格

| 项目 | 黄品 | 玄品 | 地品 | 天品 |
|---|---:|---:|---:|---:|
| 购买卡牌 | 50 | 80 | 150 | 300 |
| 升级卡牌 | 30 | 50 | 80 | 150 |

删除卡牌费用按本局已使用次数递增：50、75、100、125……每次进入黑市最多删除 1 张；牌组只剩 1 张时不可删除。

## 三、背包物品规则

背包上限为 `GameState.BAG_CAPACITY = 10`。三类物品继续使用 `elixir` / `talisman` / `formation`，以兼容事件系统的 `has_type:*` 和 `consumable_random`。

每个物品都有两种用途：

| 场景 | 设计方向 | 消耗规则 |
|---|---|---|
| 地图使用 | 补给、路线、下次黑市、下次奖励、少量下场战斗准备 | 使用后立即消耗 |
| 战斗使用 | 即时救场、爆发、抽牌、状态处理 | 使用后立即消耗；BattleScene 背包 UI 后续接入 |

阵法改为一次性“阵盘”，不再激活常驻阵法，也不再写入可反复使用的 `active_formation_id`。

## 五、实现接口

| 接口 | 说明 |
|---|---|
| `ShopDatabase.generate_stock(floor, owned_artifact_ids, seed, extra_item_count = 0)` | 返回 `{cards, items, artifacts}`；`extra_item_count` 用于下次黑市加货 |
| `ShopDatabase.get_all_items()` | 返回 25 个背包消耗品 |
| `GameState.use_consumable(index, "map")` | 应用 `map_use` 并消耗物品 |
| `GameState.consume_pending_battle_consumable_effects()` | 战斗初始化读取并清空地图登记的下场战斗效果 |
| `GameState.consume_pending_shop_bonuses()` | 黑市读取并清空下次折扣与加货 |
| `GameState.consume_pending_reward_bonuses()` | 奖励页读取并清空下次灵石/稀有度加成 |
| `BattleEngine.apply_consumable_effect(effect)` | 战斗物品逻辑钩子；战斗 UI 后续接入时调用 |

## 六、测试与手动验证

自动化：

- `TestShopDatabase`：物品总数 25、三类物品、双用途字段、库存数量、加货参数、价格、宝物去重。
- `TestShopTransactions`：地图使用回血、下场战斗效果、下次黑市、下次奖励、风符改选下一层节点、阵盘消耗。
- `TestBattleEngineLogic`：战斗物品钩子处理伤害、抽牌、护体、道行、清除负面、延后行动、濒死保护。
- `TestRewardScreen`：下次奖励灵石加成与卡牌奖励稀有度保底。

手动验证：

- 从地图点击黑市节点进入 `scenes/Shop.tscn`，确认物品货架默认 4 个。
- 使用观星阵盘后进入黑市，物品货架变为 5 个。
- 使用玄龟阵盘后进入黑市，价格显示临时 -20%。
- 使用风符·逐云后，下一次选择地图节点时，可从下一层所有同层节点中选择 1 个。
