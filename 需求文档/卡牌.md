# 《无尽仙台》卡牌手册 Ver 8.0

---

## 一、双资源体系说明

| 资源 | 每回合规则 |
| :--- | :--- |
| **灵力** | 回合开始回复【灵力回复】值；回合结束完整保留，可跨回合积累；不可超过【灵力上限】 |
| **道慧** | 回合开始重置至【道慧上限】；回合结束清零，花不完等于浪费 |

---

## 二、牌库机制

- 玩家牌库额定 **20 张**（耗尽牌除外）
- 抽牌堆归零时：弃掉所有手牌 → 与弃牌堆全部洗匀 → 重新抽 5 张

---

## 三、英雄技能系统

每回合默认限用 **1 次**，独立于出牌之外。

| 英雄 | 技能名 | 费用 | 效果 |
| :--- | :--- | :--- | :--- |
| **程天锋** | 剑意凝神 | 道慧 6 | 获得 1 层【道行】 |

---

## 四、状态词条速查

| 词条 | 类型 | 效果 |
| :--- | :--- | :--- |
| **道行** | 正面·叠层 | 每层使本次伤害 +1，跨回合保留 |
| **护体** | 正面 | 抵消等量伤害，跨回合永不归零 |
| **不侵** | 正面 | 受到伤害 -50% |
| **心流** | 正面 | 本回合伤害 +50% |
| **裂伤** | 负面 | 受到伤害时额外承受 50% |
| **枯竭** | 负面 | 术法伤害 -50% |
| **震慑** | 负面 | 本回合道慧上限 -3 |

---

## 五、卡牌词缀说明

| 词缀 | 机制 |
| :--- | :--- |
| **耗尽** | 打出后永久移出本局，不进弃牌堆 |
| **溃散** | 回合结束时若仍在手牌，自动耗尽 |
| **本源** | 战斗开始时必定出现在初始手牌 |
| **禁锢** | 无法从手牌主动打出 |
